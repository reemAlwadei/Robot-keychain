# Robot Keychain Design

This repository contains the 3D printable STL file and documentation for a custom robot keychain designed using Onshape.

---

## Designer Information

* Designed by: Reem Alwadei
* University:King Abdulaziz University
* Major: physics

---

## Technical Specifications

The design strictly follows the specified technical requirements:
* Base Thickness: 2 mm
* Keyring Hole Diameter: 4 mm
* Design Approach: Multi-part modeling created using individual feature extrusions to ensure high detail and structural integrity.
